Grammar Editor Pro (Catania Edition)

Grammar Editor Pro è uno strumento per l’ apprendimento di grammatiche context-free (grammatica libera dal contesto o CFG).
Una Grammatica è una notazione formale con cui esprimere la sintassi di un linguaggio.
Essa stabilisce delle regole come in una sorta di “gioco” in cui tutte le partite iniziano con un simbolo e lo svolgimento consiste nell’applicare le regole per generare frasi del linguaggio descritto dalla grammatica.
Grammar Editor Pro presenta un’ area di testo, “Grammatica”, in cui è possibile definire le regole della grammatica.
Dopo aver cliccato su “Genera” apparirà una nuova finestra in cui occore selezionare una delle produzioni visualizzate per ottenere la stringa finale.
Inoltre un click su “Albero” permette la costruzione automatica di un albero sintattico che mostra le regole utilizzate durante il riconoscimento dell’ input.
Grammar Editor Pro richiede la piattaforma Java 1.7
